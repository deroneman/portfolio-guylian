## 🎮 Fonctionnalités du Jeu

### HUD (Head-Up Display)
- **HP/MP**: Points de vie et de mana (régénération automatique)
- **Level Bar**: Barre de progression
- **Quest Log**: Affichage de la quête actuelle
- **Playtime Counter**: Temps passé sur le portfolio

### Système de Navigation
- Navigation par onglets (sections)
- Raccourcis clavier:
  - **P**: Aller au Profil
  - **C**: Aller aux Compétences
  - **E**: Aller à l'Expérience
  - **Espace**: Utiliser une potion (régénération complète HP/MP)

### Effets Visuels
- **Glitch Effect**: Effets visuels cyberpunk
- **Glow Effects**: Auras lumineuses
- **Particle Effects**: Effets de particules
- **Animations**: Transitions fluides
- **Mouse Tracking**: Effets de lumière suivant la souris

### Système de Mana & Potions
- **Consommation de Mana**: 
  - 10 mana par changement de catégorie
  - 5 mana par fermeture de panel
  - 0 mana pour le déplacement des panels
- **Cooldowns Séparés**:
  - Potion de Santé: 15 secondes
  - Potion de Mana: 10 secondes
- Régénération automatique du mana
- Alerte visuelle si mana insuffisant

### Système de Profil Cascadé
- 6 panels affichés en cascade (haut en bas)
- Espacement automatique de 5px entre panels
- Repositionnement automatique à la fermeture d'un panel
- Chaque panel draggable et refermable

### Persistance de Page
- Save automatique de la catégorie sélectionnée
- Récupération de la dernière page visitée au rechargement
- État du bouton menu restauré correctement

### Réactivité & Device
- **Desktop**: Zoom 90% pour affichage optimisé
- **Tablette (768px)**: Layout adapté avec panel heights: 72vh standard, 76vh special
- **Mobile (480px)**: Layout full-screen avec panel heights: 67vh standard, 71vh special
- Scrollbars cachées visuellement (scroll fonctionnel)
- Panel standard: 77vh | Experience/Projects/Organisation/Contact: 81vh

## 🚀 Comment Utiliser

### Ouvert en navigateur
1. Ouvrez le dossier `portfolio-mmorpg` dans VS Code
2. Utilisez l'extension "Live Server" pour servir les fichiers
   - Clic droit sur `index.html` → "Open with Live Server"
3. Ou simplement double-cliquez sur `index.html` pour ouvrir dans votre navigateur

### Localhost
```bash
# Si vous avez Python 3
python -m http.server 8000

# Puis accédez à http://localhost:8000
```
